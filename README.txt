Installation:
=========
*Required "youtube-dl" and "ffmpeg" package installed.
You can read here: https://rg3.github.io/youtube-dl/
-------------------
1. Download mp3-mp4.tcl

2. Open "mp3-mp4.tcl" file edit this lines:
set linkdl "http://your.link/~user"	(Change to your link patch)
set path "/home/user"	(Change to your user patch)

3. Upload mp3-mp4.tcl to your scripts folder. (Ex: /home/user/eggdrop/scripts/here)

4. Load your bot.
